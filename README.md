# ai-agents-intro

Structure du projet :

- `agents/` : agents d'explication, de création d'exercices et de relecture.
- `tools/` : utilitaires d'écriture de fichier et de validation.
- `data/topic_examples.json` : données d'exemple.
- `main.py` : génère `output/study_guide.md`.

## Lancer

```bash
python main.py
```
